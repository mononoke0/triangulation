# Deluanay triangulation
Delaunay triangulation without library. 
### 実行方法
- pointer_ope.pyがあるdirectoryで "python pointer_ope.py"
### ファイル
- first_data.py  
表面(地図)上にその地点に対応する温度を張り付ける
- pointer_ope.py  
三角形分割で使用される頂点の追加・削除・移動を行う際に, マウスの動きを受け取る
- triangulation.py  
三角形分割を行う
